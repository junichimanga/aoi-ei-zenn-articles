# Instagramを30日ほぼ触らずAIに任せた実録【無料サンプル2個公開】

## この記事でわかること

- HeyGen・Whisper・Cloudinaryを繋いだ自動投稿パイプラインの実費と構成
- 30日間で実際に「詰まった日」と、コードを書かずに回避できた手順
- コードを書けない人がどこまで真似できるか、正直な線引きの存在

---

## 「毎回ゼロから投稿する」が時間を奪い続けている

ネタを考えて、撮って、キャプションを書いて、ハッシュタグを調べて、投稿する。

それを毎日繰り返している人が多い。普通だと思っているから、疑わない。

でも考えてほしい。そのうち「機械に任せられる工程」はいくつある？

台本を書く5分だけ残して、あとをAIに流す。30日間、そのパイプラインを動かした。再生数・詰まったポイント・コスト・失敗した日の原因を省略せず記録した。

「コードが書けない自分には無理」と思うかもしれない。正直に言う。一部は無理。でも一部は真似できる。どこで線が引かれるか、先に確認しておくのが賢い。

---

## 【サンプル①】使ったツールと月の実費

30日間のパイプラインで使ったツールと、実際にかかった費用の一覧。

| ツール | 役割 | 費用（30日） |
|---|---|---|
| HeyGen（Creatorプラン） | AIアバター動画の生成 | 年間契約内に含む |
| OpenAI Whisper API | 音声→字幕タイムスタンプ生成 | 約30円（30本分） |
| Cloudinary | 動画ホスティング・URL発行 | 無料枠内（0円） |
| Instagram Graph API | Reel自動投稿 | 無料 |
| WordPress | 投稿アーカイブ・下書き保存 | サーバー代の範囲内 |

**30日の追加実費：約30円（筆者の環境・条件）。**

HeyGenはもともと動画制作に使っていたプランの範囲内。自動投稿のために追加でかかったのは、WhisperのAPI料金のみだった。1本あたり90秒前後の動画を30本処理した結果がこの数字。環境や動画の長さによって変わるため、参考値として見てほしい。

1日あたりの作業時間も実測している。

| 作業内容 | 時間 |
|---|---|
| 台本を書く（プロンプト使用） | 約5分 |
| HeyGenで動画生成ボタンを押す | 約2分 |
| パイプライン起動（コマンド1行） | 約1分 |
| 投稿確認・微修正 | 0〜5分 |
| **合計** | **約8〜15分/日** |

「ほぼ触らず」は厳密に言えば「毎日8〜15分だけ触った」が正確な表現。台本を人間が書かないとキャラが崩れる。そこだけは自動化しなかった。

## 【サンプル②】詰まった日と、コードなしで直した手順

### Day 3：字幕ファイル生成エラー

**何が起きたか**

WhisperへのAPIリクエストは通った。でもタイムスタンプが正常に返ってこなかった。字幕なし動画が生成されそうになって、手動で止めた。

**コードなしで回避した手順**

① HeyGenのWeb画面から動画を手動ダウンロード  
② 無料オンラインツール「Clideo」でMP4をいったん再変換  
③ 変換後のファイルでパイプラインを再実行 → 正常に字幕生成  

この日だけ投稿が1日ズレた。ストーリーズで「明日投稿します」と手動告知。ツールのトラブルは必ず起きる。「起きたときどう動くか」を先に知っておくと、焦りが消える。

---

### Day 25：HeyGenトークン切れ

**何が起きたか**

パイプラインがHeyGenから動画URLを取得しようとして失敗。自動投稿が止まった。

**直した手順**

① HeyGenのWeb画面にログイン  
② 設定 → API → 新しいAPIキーを発行  
③ `.env`ファイルの該当行を書き換え  
④ パイプラインを再起動  

作業時間：約5分。以後、月1回の定期チェックをカレンダーに登録した。

## このサンプルをどう活かすか

費用表はそのまま「自分の環境に当てはまるか」の確認に使える。HeyGenのプランをまだ持っていない場合は、追加費用の試算の出発点にしてほしい。

トラブルログは「事前の心構え」として読む使い方が個人的にはおすすめ。パイプラインを動かし始めると、必ず1度は止まる日が来る。Day 3とDay 25の対処手順は、コードを書けない人でも再現しやすい手順に絞って選んだ。

「コードを書ける人が有利な工程」と「書けなくても回避できる工程」の明確な線引きは、完全版の第4章で詳しく分けて記している。
