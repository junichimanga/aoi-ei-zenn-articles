## SNSを続けられない原因は「才能」じゃない

「毎日投稿しよう」と決意した3日後に止まる。
この経験、心当たりがある人は多いはず。

続かない理由は、だいたい3つに絞られる。

- **「今日、何を投稿すればいいかわからない」**
- **「文章を考えるのに時間がかかりすぎる」**
- **「管理が面倒になって、そのまま放置」**

これは才能や根気の問題ではない。仕組みの問題。

「何を書くか」を毎回ゼロから考えているから消耗する。
「書いた下書きをどこで管理するか」が決まっていないから散らばる。
「投稿日をどう決めるか」がないからペースが崩れる。

この3つを解決する仕組みを、コード不要でNotionとChatGPTだけで作れる。
そのための設計図とプロンプトをまとめたのが今回紹介するテンプレ集だ。

## 基本の運用フロー

全体の流れはシンプルに4ステップ。

```
STEP 1：ネタ出し
　ChatGPTにプロンプトを貼ってネタ候補10本を出力
　→ NotionのネタバンクDBに貼り戻す

STEP 2：構成決め
　ネタ1件を選び、構成プロンプトで「冒頭・本論・CTA」の型を生成

STEP 3：投稿文生成
　構成をChatGPTに渡して投稿文に変換
　（トンマナ・文字数・絵文字の有無を指定）

STEP 4：スケジュール登録
　Notionのカレンダービューに投稿日・媒体をセット
　→ 当日そのまま貼って投稿
```

これを**週1回・月曜の60分**でまとめてやる。「毎日考える」という消耗パターンを、週1の「まとめ仕込み」に変える。筆者の体験として、この切り替えだけでコンテンツを出し続けやすくなった。

## 【無料サンプル①】Notion ネタバンク DB の設計

まずNotionにデータベースを1つ作る。プロパティの設計が全体の使い勝手を左右する。以下をそのまま複製して使える。

### ネタバンク DB プロパティ一覧

| プロパティ名 | タイプ | 用途 |
|---|---|---|
| タイトル | テキスト | ネタの一言タイトル |
| ジャンル | セレクト | ノウハウ／日常／ツール紹介／Q&A／Before-After |
| SNS媒体 | マルチセレクト | Instagram／X／Threads／TikTok |
| ステータス | セレクト | ネタのみ／構成済み／文章完成／投稿済み |
| ネタ詳細 | テキスト（長文） | ChatGPTが出した候補をそのまま貼る欄 |
| 投稿予定日 | 日付 | カレンダービューで管理 |
| 反応メモ | テキスト（長文） | 投稿後のいいね数・保存数・コメントを記録 |

### カレンダービューの設定（30秒）

1. データベース右上「追加ビュー」→「カレンダー」を選択
2. 「日付プロパティ」で「投稿予定日」を選択

これで月カレンダーに投稿予定が一覧表示される。週3投稿なら3枠を埋めるだけ。投稿枠が視覚化されることで、ペースを意識しやすくなる。

### コンテンツカード内のセクション構成

各レコードを開いたページ内に、以下のセクションを**Notionのテンプレートブロック**として保存する。設定は「DB右上の…→テンプレート→新しいテンプレート」で以下を貼り付けるだけ。

```
## ネタ概要
（ChatGPTが出したネタ候補をここに貼る）

## 構成メモ
・冒頭フック：
・本論ポイント：
・CTA（行動を促す一言）：

## 投稿文（完成版）
（ChatGPTが出した投稿文をここに貼る）

## ハッシュタグ
（生成されたハッシュタグをここに貼る）

## 反応記録
　いいね：　保存：　コメント：　リーチ：
```

新しいレコードを追加した瞬間にこの構成が自動展開されるため、毎回ゼロから書く手間がなくなる。

## 【無料サンプル②】ネタ出しプロンプト：ジャンル指定で10本一括生成

ChatGPT・Claude・Gemini いずれにも使える。`【】`の中だけ自分の情報に書き換えて送信するだけ。

```
あなたはSNSコンテンツの企画専門家です。

以下の条件でInstagram投稿のネタを10本提案してください。

【発信ジャンル】：AIツール活用・業務効率化
【ターゲット読者】：副業中の会社員・AIを使ったことない初心者
【避けたい内容】：専門用語・コードや開発の話

各ネタは以下の形式で出力してください：
・タイトル（25文字以内）
・一言概要（40文字以内）
・なぜ刺さるか（読者目線で30文字以内）
```

### 出力イメージ（例）

```
① AIに明日の仕事を頼む方法
　概要：ChatGPTに「明日のタスクリスト」を作ってもらう具体手順
　刺さる理由：明日の準備ができていない不安を解消できるから

② スマホ1台で議事録を自動化
　概要：会議録音→Whisper→要約まで10分でできる
　刺さる理由：会議後の議事録作業が苦手な人に刺さる

（以下10本出力される）
```

この出力をそのままNotionの「ネタ詳細」欄に貼る。ステータスを「ネタのみ」にして保存。以後、このネタ一覧から週3本を選んで投稿文に育てていく流れになる。

## サンプルのカスタム方法

プロンプト②はジャンルとターゲットを変えるだけで、どんな発信領域にも使い回せる。

- `「ファッション・40代向けコーデ」`
- `「子育て・0〜3歳の親向け時短術」`
- `「読書・ビジネス書を月10冊読む人向け」`

Notionの「ジャンル」セレクト選択肢を自分の発信領域に合わせて書き換えれば、DB全体が自分仕様になる。「SNS媒体」のマルチセレクトでカレンダービューを絞り込み表示することもできるため、複数媒体を並行運用している場合にも対応できる設計になっている。
