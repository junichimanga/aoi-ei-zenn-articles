# Notion×ChatGPTで作るSNS30日ネタ出しテンプレ【無料サンプル3個公開】

## この記事でわかること

- 「今日何を投稿しよう」という消耗ループを仕組みで断ち切る方法
- ChatGPTから使えるSNS投稿案を引き出すプロンプトの構造と書き方
- Notionをコンテンツカレンダーとして機能させる考え方

---

## 毎回ゼロから考えている、その時間は取り戻せない

SNS運用で多くの人が最初につまずくのは、「継続できない」ではなく「ネタが尽きる」だ。

投稿の内容自体は悪くない。文章も書ける。でも「今日何を発信しようか」と考え始めた瞬間に30分が消え、結局ありきたりな内容を投稿して終わる——この繰り返しに心当たりがある人は少なくないはずだ。

問題は意欲でも才能でもなく、**ネタ出しが「毎回即興」になっていること**にある。

料理に例えると、食材も調理法も決まっていないのに「今晩のメニューを考えながら買い物して、作る」ようなものだ。仕込みがあれば話は別だが、現状はほぼ全員がぶっつけ本番でやっている。

この記事では、Notionにプロンプトを組み込んでコンテンツカレンダーをセミオートで回す仕組みの考え方と、実際に使えるプロンプトサンプルを3本公開する。

---

## 仕組みの全体像

構成はシンプルだ。

```
Notionデータベース
└─ 曜日列 / ジャンル列 / フォーマット列 / プロンプト列 / 投稿案列
```

「月曜はハウツー型・火曜はフック型・水曜は問いかけ型」のように曜日とフォーマットをあらかじめ決め打ちしておく。各行に対応するChatGPTプロンプトを貼り付け、そのまま実行して投稿案を「投稿案列」に貼るだけ。

一度テンプレを作ってしまえば、**毎日やることは「プロンプトを実行して投稿案をコピーするだけ」**になる。

では実際のプロンプトを3本見ていこう。

## 無料サンプル3本

### サンプル①：フック型クロス媒体変換プロンプト

1つのアイデアをInstagram・X・Threads向けに同時変換する。3媒体を使い分けている人に特に有効だ。

```
以下のアイデアをSNS投稿ネタに変換してください。

【アイデア】
{{ここに1〜2行で伝えたいことを入力}}

【出力形式】
媒体ごとに以下を出力してください。

■ Instagram（150〜200字）
・冒頭1行でスクロールを止めるフック文から始める
・体言止めや短文を活用し、テンポよく読める文体
・最後にフォロワーへの問いかけか行動促進の一文

■ X（〜140字）
・論点を1つに絞る
・「〇〇な人は読んで」「実は〇〇だった」などの引きを入れる
・リプライを誘発する終わり方

■ Threads（100〜150字）
・日常会話に近いトーン
・読者が「わかる」と感じる言葉を選ぶ
・コメントしやすい問いかけで締める

【制約】
・誇張表現・断言は使わない
・専門用語は使わず、読んでわかる言葉のみ使う
```

---

### サンプル②：Instagram用3ステップ解説プロンプト

複雑な内容を「3ステップ」に圧縮することで、保存率が上がりやすい解説投稿を生成できる。

```
以下のトピックについて、Instagramの解説投稿ネタを生成してください。

【トピック】
{{解説したい内容・ツール・スキル名を入力}}

【対象読者】
{{例：AIツールを使い始めたばかりの社会人、副業に興味がある30代など}}

【出力形式】

■ キャプション冒頭（フック文、1〜2行）
→ 「〇〇してる？」「知らないと損する〇〇」などスクロールを止める書き出し

■ 3ステップ本文
Step1：〇〇する（1〜2行で説明）
Step2：〇〇する（1〜2行で説明）
Step3：〇〇する（1〜2行で説明）

■ まとめ文（1〜2行）
→ 行動を促すか、感想を求める締め

【制約】
・1ステップは2行以内に収める
・数字や記号（→、◆）でテンポよく読めるようにする
・「最高」「必ず」「100%」などの絶対表現は使わない
```

---

### サンプル③：悩み共感型問いかけプロンプト

ターゲットの潜在的な悩みに触れる問いかけ投稿は、フォロワーとの距離を縮めやすい。エンゲージメント目的の投稿に使いやすいフォーマットだ。

```
以下の情報をもとに、SNS用の「共感型問いかけ投稿」を生成してください。

【ターゲットが抱えている悩み】
{{例：SNS投稿が続かない、ネタが思い浮かばない、何を発信すればいいかわからない}}

【あなたのジャンル・テーマ】
{{例：AIツール活用、副業、Notionなどの生産性ツール}}

【出力形式】

■ 冒頭の問いかけ文（1〜2行）
→「〇〇で悩んだことある？」「〇〇になったとき、どうしてる？」など共感を引く問い

■ 自分の経験・気づき（3〜5行）
→ 同じ悩みを持っていたこと、どう対処したかをリアルに書く
→ 「私は〇〇でした」など一人称で語る

■ 読者への問いかけ（1行）
→ コメントしたくなる質問で締める

【制約】
・上から目線にならないよう、あくまで経験談ベースで語る
・共感を引く言葉（「わかる」「正直」「ぶっちゃけ」など）を自然に入れる
・投稿全体は200字以内に収める
```

## 使い方とカスタム方法

Notionデータベースに「フォーマット」列を作り、hook・howto・questionのいずれかを入力する。「曜日」列と掛け合わせてフィルタを設定しておくと、「今日のフォーマット」が自動で絞り込まれる仕組みになる。プロンプトの`{{　}}`部分をNotion上の他の列と連携させれば、入力→実行→コピーの流れがほぼルーティン化できる。サンプルのプロンプトはそのまま使ってもいいし、自分のジャンルや口調に合わせて制約部分だけ書き換えるのが最も手早い。
